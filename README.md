## Featured Projects

### 学校ポータルアプリ

学生が必要な情報をスマートフォンから確認できるポータルアプリです。

- Repository: 準備中
- 使用技術: Flutter, Supabase, PostgreSQL
- 機能: 電子学生証、出欠確認、時間割、FAQ、学習記録
- 目的: 学校生活に必要な情報を一つのアプリに集約すること
- 工夫: スマホファーストで、学生が迷わず使える情報導線を意識
- 今後: NFC出席、通知機能、FAQチャットボットを追加予定

### 出欠管理システム

授業の出欠を管理するWebシステムです。

- Repository: 準備中
- 使用技術: HTML, CSS, JavaScript, PHP, MySQL
- 機能: 出欠登録、学生別確認、出席率計算
- 目的: 手作業での出欠管理を効率化すること
- 工夫: 登録・確認・修正の流れを分かりやすく整理

### 防災アプリ

地域の災害情報や避難支援を目的としたアプリです。

- Repository: 準備中
- 使用技術: Flutter, Supabase
- 目的: 必要な災害情報を分かりやすく届けること
- 想定機能: 避難情報、雨量・河川情報、マップ連携、自治体情報への導線
